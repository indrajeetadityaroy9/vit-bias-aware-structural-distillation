# BASD: Bias-Aligned Spectral Distillation

Feature-based knowledge distillation for Vision Transformers remains an open problem: fixed layer mappings ignore the evolving geometry of student representations during training, and multi-loss frameworks introduce millions of auxiliary parameters that compete with the student for gradient signal. This work introduces BASD, an architecture-agnostic distillation method built on a single principled mechanism — a spectrally-adaptive Grassmannian layer selector that introduces only four learnable parameters. At each training step, BASD projects teacher and student intermediate representations onto the Grassmann manifold, estimates per-layer subspace rank via the Marchenko-Pastur threshold from random matrix theory, and computes spectrally-weighted principal angles that emphasize high-energy directions in the teacher's singular value spectrum. These distances produce soft mixing weights over all teacher layers, yielding a training-adaptive alignment target that tracks the student's evolving representational structure. The mixed teacher tokens are aligned to the student spatial grid through parameter-free interpolation and matched via an attention-weighted Procrustes loss that handles arbitrary dimension mismatch through rectangular SVD of the cross-covariance. Inverse-loss weighting (UW-SO) automatically balances the Procrustes objective against cross-entropy without tuning. BASD unifies ViT-to-ViT and CNN-to-ViT distillation under one interface, treating CNN features as single-layer tokens with uniform attention, and requires no architecture-specific projectors, head aligners, or loss toggles.
